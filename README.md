# Ai-Driven-Study-Planner-System-
An AI-powered study planning system that generates personalized schedules, tracks progress, and optimizes learning based on user performance.
Hope you would benfiit yourself whole from this project :)


This project is an AI-assisted offline study planning system that helps students automatically generate daily and weekly study schedules.
The tool uses rule-based AI logic to calculate recommended study hours, distribute subject workloads, and create structured timetables without requiring an internet connection.

The system includes:

A Daily Study Planner

A Weekly Study Planner

A Home Dashboard

Offline logic for predictions & schedule generation

Fully responsive UI built from scratch without frameworks
